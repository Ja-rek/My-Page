| Suckless.assert

A library for expressing side effects by creating a contract for beginning value (arguments). This technique can be referred to as ”design by contract,” ”defense programming,” or ”guard methods”. The Suckless Assert is much faster and simpler than other similar libraries that support fluent API and can get a field/property name by the lambda expression.
